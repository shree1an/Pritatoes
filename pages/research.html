Development Of Home Page

Reference Home Page

Pritatoes is inspired by the home page of the Coco Ross Design, which features a clean and elegant design. We took this inspiration and implemented it into our own website home page by creating a simple and modern design with a focus on showcasing our brand and other menu content. We also included a featured products section and about our ecommerce website. To further help our customers, we included a blog section where customers can find helpful tips, articles, and other resources to help them learn more about the products we offer. Additionally, we have also included social links, making it easy for customers to stay connected and up-to-date on the latest news and products.

Similarities

  • Elegant navigation bar of Coco Ross Design is similar to the Pritatoes as it has similar white background navigation bar with border

  • Elegant front view is similar to the Pritatoes first view as it have similar division of image and content with button.

Difference

  • Above Navigation bar is different as Pritatoes have contact info and link to the social media.

  • Pritatoes have link to the different printer company websites in logo of the brand.

Development Of Products Page

Reference Products Page

Pritatoes is inspired by the grid structure used by PANTUM. Our product page gives a detail description of the product along with the high quality image of the product. This gives the customer an idea on how the products looks in real-life. Also inspired by the hover effects of the products card which is user friendly and easy to pick.

Similarities

  • Hover effect of the products are similar

  • Grid structure for placing the products are similar.

Difference

  • Pritatoes products page have add to cart button with the pop-up function on click.

  • Products actual price and discounted price are given for user friendliness.

Development Of Blog Page

Reference Blog Page

Pritatoes Blog Page is inspired by the Daraz Blog because fo the simple grid design and main post on the top which makes it easy to find different type of blogs.

Similarities

  • Grid design of the blog are similar.

  • Blogs are featured in similar way, as both have imgae above the anchor link

Difference

  • Pritatoes have readmore button whereas Daraz blog don't.

  • Pritatoes does not consist of any advertisement. Only contains useful blog for user benifit.

Development Of Go To Top

Reference Go To Top

Go to top button is inspired from Alibaba because it is easier to click button and scroll to the top then scrolling all the long page manually.

Similarities

  • Similar position is used for the button because of the visual memories of the user as most of the website have this in same position

  • Hover effects is used and button disappear when it reaches top.

Difference

  • Different approach of the button is used as Pritatoes have word vertically placed for the button rather than symbol

  • Smooth behavior of scrolling is used to enhanche user experience.

Development Of Cart

Reference Side Cart

Side Cart is inspired from Allbirds, which allows user to quickly view the items and easily add remove or adjustquantities of the items without navigating to a different page which enhanches the overall user experience and save times.

Similarities

  • Same postion is used to not affect the user visual memories of using cart.

  • Same approach of increasing quantity and updating price is used.

Difference

  • Less content is used to make things simple to navigate and enhancing user experience

  • No blur effect is used for user to add products to cart without closing cart.
